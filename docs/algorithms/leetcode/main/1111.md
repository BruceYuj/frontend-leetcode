## 简介
- [1111. 有效括号的嵌套深度](https://leetcode-cn.com/problems/maximum-nesting-depth-of-two-valid-parentheses-strings/)

## 解法一 - 栈
我们用栈维护已经读取的左括号，保证里面一半是 A 的，一半是 B 的。因此只要把奇数的左括号给 A，偶数的给 B 就可以了。

这里由于只有奇偶性，只需要用一个变量模拟就行了。
```javascript
var maxDepthAfterSplit = function(seq) {
    let dep = 0;
    return seq.split("").map((value, index) => {
        if (value === '(') {
            ++dep;
            return dep % 2;
        } else {
            let ans = dep % 2;
            --dep;
            return ans;
        }
    });
}
```

**复杂度分析**：
- 时间复杂度：$O(n)$
- 空间复杂度：$O(1)$

## 解法二 - 找规律
因为题目给的是有效括号。我们希望 $A$ 和 $B$ 的最大深度最小。因此，我们不希望各自的深度涨的速度不一致。因此，我们希望他两此消彼伏。

```javascript
/**
 * @param {string} seq
 * @return {number[]}
 */
var maxDepthAfterSplit = function(seq) {
    let ans = new Array(seq.length).fill(0);

    let a = 0;
    let b = 0;

    for(let i = 0; i < seq.length; i++) {
        if(seq[i] === '(') {
            if(a > b) {
                ans[i] = 1;
                b++;
            } else {
                a++;
            }
        } else {
            if(a > b) {
                a--;
            } else {
                b--;
                ans[i] = 1;
            }
        }
    }

    return ans;
};
```

**复杂度分析**：
- 时间复杂度：$O(n)$
- 空间复杂度：$O(1)$
