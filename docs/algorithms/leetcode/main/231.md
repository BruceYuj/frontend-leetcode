## 思路
### 解法一 - 暴力法
按照幂次运算的定义来计算:$2^k = 2*2*2...*2$
因此，我们将 $n$ 不断的取余看余数是否等于 $0$。
```javascript
var isPowerOfTwo = function(n) {
    if(n < 1) return false;
    while(n%2 === 0) {
      n = n/2;
    }
    return n === 1;
};
```

**复杂度分析**:
- 时间复杂度：$O(log_2(n))$
- 空间复杂度：$O(1)$

### 解法二 - 位运算
目前主流的编程语言编码都是采用的补码的形式：
`-x = ~x+1`

而如果 $x$ 是 $2$的幂次数，那么 $x$ 表示为2进制一定是: $100000000...00000$，那么 `~x+1` 仍然等于$x$。

也就是 `x&-x == x`

```javascript
var isPowerOfTwo = function(n) {
    if(n < 1) return false;
    return (n & -n) === n; 
};
```
**复杂度分析**:
- 时间复杂度：$O(1)$
- 空间复杂度：$O(1)$


### 解法三 - 位运算
如果 $x$ 是 $2$的幂次数，那么 $x$ 表示为2进制一定是: $100000000...00000$，那么 `x & (x-1)` 等于$0$。

``` javascript
var isPowerOfTwo = function(n) {
    if(n < 1) return false;
    return (n & n-1) === 0; 
};
```
**复杂度分析**:
- 时间复杂度：$O(1)$
- 空间复杂度：$O(1)$
