## 解法一 - 快慢指针
1. 设定快慢指针 fast 和 slow，都初始化为 head
2. fast 每次走两步，slow 每次走一步，直到 fast 走到链表结尾

```javascript
var middleNode = function(head) {
    let slow = head;
    let fast = head;

    while(fast && fast.next) {
        fast = fast.next.next;
        slow = slow.next;
    }

    return slow;
};
```

**复杂度分析**: 
- 时间复杂度： $O(N)$， $N$ 是链表长度
- 空间复杂度： $O(1)$
