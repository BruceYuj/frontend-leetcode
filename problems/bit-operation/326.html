# 简介

-326. 3的幂

# 思路

# 解法一 - 暴力法

不断对 n 进行取余，来看其是否等于 0 。

var isPowerOfThree = function(n) {
    if(n < 1) return false;

    while(n % 3 === 0) {
      n /= 3;
    }
    return n === 1;
};

# 解法二 - 数学+计算机限制

因为通常计算机能够表示的整数是有限的（不考虑大数表示）。比如 32-bit integer。因此，我们可以提前将所有的 3 的幂全部缓存起来。通过数学计算， 中 总共有 20 个 3 的幂。

var isPowerOfThree = function(n) {
    let map = {};
    let helper = 1;
    for(let i = 0; i < 20 ; i++) {
      map[helper] = true;
      helper *= 3; 
    }
    return map[n] === true;
};
